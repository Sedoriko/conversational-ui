Marek Malík Marek Malík se zabývá oblastí User Experience a jeho dílčími obory, jako jsou informační architektura, interakční design, UI design a další. Conversational UI ho zajímá především z pohledu UX a protože vnímá, že se tímto směrem ubírá trend budoucnosti. Kryštof Davídek Kryštof Davídek se o problematiku Conversational UI dříve nezajímal, a proto je pro něj tohle téma výzva, každopádně fenomén dialogových systémů bere jako zajímavou součást aktuálních technologií a rád se o tomto tématu dozví prostřednictvím toho tématu více.
